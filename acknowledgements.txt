This project is a port from the Java zmanim-project (http://www.kosherjava.com/zmanim-project/) developed by Eliyahu Hershfeld.
----

The Zmanim Project is making use of the following Open Source projects:

* IKVM.NET is an implementation of Java for Mono and the Microsoft .NET Framework. (http://www.ikvm.net/)
* Newtonsoft.Json from James Newton-King (http://james.newtonking.com/pages/json-net.aspx)
* Quartz.NET is a full-featured, open source job scheduling system that can be used from smallest apps to large scale enterprise systems. (http://quartznet.sourceforge.net/)
* NDesk.Options (http://www.ndesk.org/Options)
* Topshelf is a lightweight framework for building Windows services using the .NET framework. The key goal of Topshelf is limiting the surface area of service-related code within the application. (http://code.google.com/p/topshelf/)
* TweetSharp is a complete .NET library for microblogging platforms that allows you to write short and sweet expressions that convert automatically to web queries and fly to Twitter on your behalf. (http://tweetsharp.com/)
